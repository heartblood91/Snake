# Description

Cette application permet de jouer au célèbre jeu snake, popularisé par le Nokia 3310 en 1998

Langage utilisé:

- CSS
- HTML
- Javascript

# Installation

```shell
$ git clone "https://github.com/heartblood91/snake.git" && cd snake\
$ npm i
$ npm run start
```
